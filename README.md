# Continuum

Mastery over time for your Ecto models

## Installation

The package can be installed as:

  1. Add continuum to your list of dependencies in `mix.exs`:

        def deps do
          [{:continuum, "~> 0.0.1"}]
        end

  2. Ensure continuum is started before your application:

        def application do
          [applications: [:continuum]]
        end
